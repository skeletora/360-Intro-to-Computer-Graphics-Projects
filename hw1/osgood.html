Problems Encountered: The main issue I encountered was the triangles deleting themselves once the divisions got to less than a pixel. In my attempts to fix it, I created some interesting types of iterations that were incorrect. For example, one time, whenever I clicked the step button, I think it was splitting the triangle, but only storing 1 of the triangles. the smaller triangles would spiral around the canvas toward the center. I eventually came up with an appropriate solution.

Interesting Techniques Employed: Not sure if my techniques are very interesting, but I found implementing the gradient fun and making the isosceles configuration.

Sources of Information: I heavily referenced Dr. Bennett's notes and examples in order to make this program. I also used websites that provided lists of html canvas commands and supported color names.

Notes on Additions: The gradient button will cause a gradient from the background color to the specified fill color and it will go from top to bottom. It defaults to off.
